# Contribution Guidelines

-No pornography or hentai in your pull request.
-No disobeying the GitHub Terms of Service.
-No racism.
-No hate.
-No false and/or useless pull requests.
#### More guidelines will be added as necessary

### Issue Guidelines:

-No racism.
-No hate.
#### More guidelines will be added as necessary.
